Quick Start

  1. Check Overview for low stock and movement activity.
  2. Manage SKUs in Items.
  3. Manage warehouse locations in Storage Areas.
  4. Apply quantity changes in Movements.

Core Workflows

Add Item

  1. Open Items and click Add Item.
  2. Fill SKU, name, category, unit, reorder level.
  3. Save.

Receive Stock

  1. Open Movements.
  2. Set type to Receive.
  3. Choose item and destination area.
  4. Enter quantity and reference like PO-1003.
  5. Apply movement.

Transfer Stock

  1. Set type to Transfer.
  2. Pick source area and destination area.
  3. Enter quantity and note.

Roles and Permissions

  • Owner: full access, including settings, audit, and permanent trash delete.
  • Manager: normal operations and data updates.
  • Viewer: read-only access.
